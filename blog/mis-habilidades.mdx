## React

Desarrollé varias interfaces de usuario con React, tecnología que considero como mi principal. Mi objetivo permanente es crear las mejores aplicaciones web posibles en cuanto a funcionalidad y calidad, por lo cual siempre me inclino hacia extender mis conocimientos de librerías y técnicas modernas que acompañan el framework. 

Mediante proyectos personales y durante mi apprenticeship en Alkemy.org, incorporé prácticas profesionales de manejo de estado (Redux, Context), integración de APIs, autenticación, testing (React-Testing-Library), estilos (css, tailwind, bootstrap), y muchas otras APIs y librerías que surgieron necesidad en los proyectos. Además de web, desarrollé apps para Android con React Native. 

También estoy familiarizado con NextJs para crear apliaciones React con pre-fetching, static-server generation  y server-side rendering. Esta web es una NextJs app desplegada en Vercel, con todos los paths generados estáticamente.

En cuanto a deployment y entorno de producción, publiqué aplicaciones tanto en Vercel como en Heroku, con variables de entorno y dominios conectados. 

## Javascript

Mediante cursos y proyectos, me he estado familiarizando con el lenguaje con una profundidad cada vez mayor.

Los conceptos básicos los aprendí en conjunto con los primeros cursos qué tomé en Udemy. Más adelante, durante la aceleración/apprenticeship en Alkemy.org, conocí los principios SOLID y mejoré considerablemente mi calidad de código. 

Últimamente estuve aprendiendo programación orientada a objetos en Javascript, y varios conceptos avanzados del lenguaje (como closures, prototipos, el motor, etc.).


## NodeJs

Desarrollé APIs para aplicaciones end-to-end con ExpressJs utilizando Mongodb para bases de datos (mongoose), y passportjs para autenticación de requests y sesiones (strategies local y externas). También implementé las SDK de Sendgrid, Firebase, MercadoPago (Checkout Pro) y AWS (para almacenamiento de imágenes en S3), entre otras.
